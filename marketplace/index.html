Submit Your Addon

Add your addon to the airlinklabs/addons repository by opening a pull request. Each addon gets its own folder.

1

Fork the addons repo

Fork airlinklabs/addons on GitHub and create a new folder with your addon's unique ID (e.g. my-cool-addon/).

2

Add info.json

{ "name": "My Addon", "version": "1.0.0", "description": "What it does", "author": "yourname", "status": "working", "tags": ["Tag1"], "icon": "https://example.com/icon.svg", "features": ["Feature 1", "Feature 2"], "github": "https://github.com/you/your-addon" }
3

Add install.json

{ "note": "Optional tip shown after install steps", "steps": [ { "title": "Clone the addon", "commands": ["git clone ... /var/www/panel/storage/addons/my-addon"] }, { "title": "Restart panel", "commands": ["systemctl restart airlink-panel"] } ] }
4

Open a Pull Request

Once merged, the registry auto-updates every 6 hours and your addon appears here.
